# Jumana Haseen Portfolio Project

## Overview
The Jumana Haseen Portfolio Project is a personal portfolio website created by Jumana Haseen to showcase her skills, projects, and professional experience. This documentation provides an overview of the project structure, dependencies, customization, and usage instructions.

## Project Structure
The project structure for the Jumana Haseen Portfolio Project is as follows:

```
JumanaHaseenPortfolio-Project/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
└── images/
    ├── project1.jpg
    ├── project2.jpg
    └── ...
```

- `index.html`: The main HTML file that serves as the entry point to the website.
- `css/style.css`: The CSS file containing custom styles for the website.
- `js/script.js`: The JavaScript file containing custom scripts for the website.
- `images/`: A directory containing project images and other related images.

## Getting Started
To view the Jumana Haseen Portfolio website locally, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/jumana07/JumanaHaseenPortfolio-Project.git
   ```

2. Open the `index.html` file in a web browser.

## Dependencies
The Jumana Haseen Portfolio Project utilizes the following dependencies:

- Bootstrap (version 5.0.0): A popular CSS framework for building responsive and mobile-first websites.

## Customization
Jumana Haseen has customized the Bootstrap template to reflect her personal style and showcase her projects and experience. The customization includes:

- Modification of the HTML structure to add relevant content and sections.
- Customization of the CSS (style.css) file to apply Jumana's preferred styles, colors, and layout.
- Utilization of the JavaScript (script.js) file to add interactive features, such as smooth scrolling and project filtering.

## Usage
To navigate and interact with the Jumana Haseen Portfolio website, follow these instructions:

1. Open the `index.html` file in a web browser.
2. Explore the different sections, including the home, about, projects, and contact sections.
3. Click on the navigation links to jump to specific sections or use the smooth scrolling feature.
4. Interact with project cards to view more details or filter projects by category.
